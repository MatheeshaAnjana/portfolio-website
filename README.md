# 🚀 Matheesha Amarathunga — Personal Portfolio

A modern, responsive personal portfolio website built with **React** and **React Bootstrap**, featuring a sleek dark theme, smooth animations, and full mobile responsiveness.

🌐 **Live Demo :** (https://matheeshaanjana.github.io/portfolio-website/)

---

## 📸 Preview

> Portfolio includes: Hero, About, Skills, Projects, Certifications, Contact, and Footer sections.

---

## 🛠️ Built With

| Technology | Purpose |
|------------|---------|
| React 18 | Component-based UI framework |
| React Bootstrap 2 | UI components (Cards, Navbar, Form, Grid) |
| Bootstrap 5 | Base CSS framework |
| CSS3 + CSS Variables | Custom dark theme & animations |
| Google Fonts | Syne + DM Sans typography |
| gh-pages | GitHub Pages deployment |

## ⚡ Getting Started

### Prerequisites
Make sure you have **Node.js** installed on your machine.
Download it from: https://nodejs.org (LTS version recommended)

### Installation & Running Locally

**1. Clone the repository**
```bash
git clone https://github.com/MatheeshaAnjana/portfolio.git
cd portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm start
```

Opens at **http://localhost:3000** in your browser. The page reloads automatically when you make changes.
